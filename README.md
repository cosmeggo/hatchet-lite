# Hatchet Lite
A balanced, stripped-down version of Hatchet

# Dependencies
- [Steamodded](https://github.com/Steamodded/smods) (Required)

This mod may not be compatible with other pre-existing mods. 

# Features
- 30 Jokers
- 10 Sephirot Cards
- 3 Enhancements
- 1 Edition
- 4 Booster Packs
- 1 Decks
- 2 Boss Blinds
